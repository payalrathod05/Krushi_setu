ğŸ›  Installation & Setup
1. Clone the Repositor
git clone https://github.com/payalrathod05/Krushi_setu.git
cd KrushiSetu/fronted (Fronted)
cd KrushiSetu/server (backend)
3. Install Dependencies
npm install
4. Start Development Server
npm run dev (Fronted)
nodemon server.js (Backend)
Your app will be available at:

http://localhost:5173/
http://localhost:3000/

4. Create Production Build
